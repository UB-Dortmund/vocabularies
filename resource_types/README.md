Resource Type Vocabulary
========================

## Changelog v1.1

### Added Translations of Labels

In addition to English, Chinese, Spanish, French, Russian, German, Catalan, Italian, Portuguese the labels are translated into the following languages:

* Dutch
* Turkish
* Japanese
